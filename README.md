# Football Match Statistics Analyzer
A Python application that analyzes football match datasets and generates performance indicators based on team statistics.

## Features
- CSV dataset processing with Pandas
- Match outcome simulation
- Shot efficiency calculation
- Aggressiveness index analysis
- Corner kick comparison
- Offensive pressure indicator (IPO)
- Team pressure simulation using statistical metrics

## Technologies
- Python
- Pandas

## Metrics Generated
- Match winner prediction
- Shot conversion efficiency
- Aggressiveness score
- Corner kick comparison
- Offensive pressure index (IPO)
- Pressure endurance simulation

## Learning Objectives
This project was developed to practice:

- Data processing with Pandas
- Statistical analysis
- Algorithm development
- Conditional logic
- Loops and simulations
- Working with structured datasets
